informatie cpu

Een programmacode uit te voeren doorloopt een CPU een aantal stappen. Het programma zelf staat in het RAM-geheugen. De CPU krijgt instucties uit het ROM geheugen. Als de CPU de instructies heeft ontvangen gaat hij aan de slag met het verwerken en uitvoeren van het programma of een code. Als de CPU dit heeft gedaan slaat hij alles op in het permanente geheugen.(Opslag zoals SSD's en HDD's) De CPU zijn snelheid wordt gemeten in GHz. Dus hoe meer GHz de CPU heeft hoe sneller hij is. Je hebt ook cores. Een core wordt gebruikt om te multitasken. Dus een CPU die meer cores heeft kan beter multitasken.
